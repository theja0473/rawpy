API Reference¶

The main entry point to start working with RAW images is rawpy.imread(). It returns a rawpy.RawPy object on which all further operations have to be performed.

rawpy.imread(pathOrFile)¶

Convenience function that creates a rawpy.RawPy instance, opens the given file, unpacks the RAW data, and returns the rawpy.RawPy instance for further processing.

Parameters:pathOrFile (str|file) – path or file object of RAW image that will be read
Return type:rawpy.RawPy
